# 01. AI 교육 플랫폼 초기 구축

작업일: 2026-06-09

## 목표
온라인 동영상 콘텐츠를 제공하는 AI 교육회사 사이트를 구축한다.  
rest03(건설사 템플릿)의 아키텍처를 기반으로 AI 교육 도메인에 맞게 전면 재설계한다.

## 참고 소스
- rest03: React + Vite + Tailwind 기반 회사 사이트 템플릿 (라우팅 구조, 컴포넌트 패턴 참고)
- 서비스 방향: 유튜브 영상 기반 AI/AI 리터러시 강의 무료 제공

## 기술 선택
| 항목 | 선택 | 이유 |
|------|------|------|
| Framework | React 18 + Vite 5 | rest03 연속성, 빠른 HMR |
| 스타일 | Tailwind CSS 3 | 유틸리티 우선, 반응형 용이 |
| 라우팅 | React Router 6 + HashRouter | GitHub Pages SPA 호환 |
| 상태 | Context API (ThemeContext) | 다크모드·팔레트 전역 관리 |
| 배포 | GitHub Actions → GitHub Pages | `main` push 시 자동 빌드·배포 |

## 컬러 팔레트
5가지 색상으로 브랜드 아이덴티티 구성:

| 이름 | HEX | 용도 |
|------|-----|------|
| Deep Navy | `#0F1B2D` | 다크 배경, 히어로 |
| Royal Blue | `#1E3A8A` | 메인 브랜드, 버튼 |
| Sky Blue | `#3B82F6` | 링크, 인터랙션 강조 |
| Emerald Teal | `#0D9488` | 보조 액센트 |
| Gold Amber | `#F59E0B` | 하이라이트, CTA |

## 다크 / 라이트 모드
- Tailwind `darkMode: 'class'` 방식
- 헤더 토글 버튼으로 전환 (Sun/Moon 아이콘)
- OS 설정 자동 감지 (`prefers-color-scheme`)
- `localStorage`에 `theme` 키로 저장 → 새로고침 유지
- `index.html` 인라인 스크립트로 깜빡임(FOUC) 방지

## 동영상 카테고리 구성
5개 주제별 카테고리, 각 탭 내비로 분리:

| 카테고리 ID | 이름 | 설명 |
|------------|------|------|
| `ai-basics` | AI 기초 | 인공지능 개념·역사·원리 (12개) |
| `ai-literacy` | AI 리터러시 | AI 시대 필수 역량 (8개) |
| `generative-ai` | 생성형 AI | ChatGPT·이미지 AI 실습 (6개) |
| `ai-tools` | AI 도구 활용 | Notion AI·Gamma 등 실무 (6개) |
| `ai-future` | AI와 미래 | AI 사회 변화·직업 (6개) |

## 유튜브 영상 그리드
- **2열 × 3행 = 6개/페이지** 레이아웃 (모바일 1열)
- 페이지네이션: 이전/다음 + 페이지 번호 버튼
- `VideoCard`: YouTube 썸네일 자동 로드, hover 시 재생 오버레이
- `VideoModal`: YouTube iframe 임베드, ESC/오버레이 클릭으로 닫기
- 영상 ID는 `src/data/site.js`의 `youtubeId` 필드에 입력 (현재 플레이스홀더)

## 구현 범위
| 영역 | 구현 내용 |
|------|-----------|
| 공통 | Header(드롭다운 메뉴, 모바일 햄버거), Footer(카테고리·연락처), ScrollToTop, ScrollToTopButton |
| 홈 | Hero(그라디언트 배경·통계), 카테고리 카드, 최신 영상 6개 미리보기, 특징 소개, CTA |
| 동영상 | 카테고리 탭, 2×3 그리드, 페이지네이션, YouTube 모달 |
| 커리큘럼 | 단계별 강의 목록, 카테고리별 로드맵 |
| 회사소개 | 회사소개·비전·팀·연혁 탭 구조 |
| 문의하기 | 폼(이름·이메일·제목·내용), 제출 완료 상태 |
| 기타 | 개인정보처리방침·이용약관 SimplePage |

## 데이터 분리
- `src/data/site.js` 중앙 데이터 관리:
  - `siteConfig`: 회사명·연락처·소셜
  - `navMenu`: GNB 구조
  - `videoCategories`: 카테고리 메타
  - `videos`: 카테고리별 영상 데이터
  - `features`, `stats`: 홈 섹션 데이터

## 모바일 최적화
- 모든 레이아웃 모바일 퍼스트 설계
- 헤더: 햄버거 메뉴 → 슬라이드 패널
- 그리드: 1열(모바일) → 2열(sm) → 3열(lg)
- 터치 대상 최소 44px 확보

## 검증
- `npm run build` 성공 (48 modules, 빌드 ~1.5s)
- 번들 크기: JS 221KB (gzip 69KB), CSS 36KB (gzip 6KB)

## 배포
- `.github/workflows/deploy.yml` 구성: `main` push → Vite build → GitHub Pages 업로드
- 배포 URL: https://eclipticwin.github.io/rest04/
- ⚠️ GitHub 저장소 Settings → Pages → Source를 **GitHub Actions**로 설정 필요
